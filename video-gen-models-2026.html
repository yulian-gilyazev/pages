Методология
AA Video Arena ELO ↗ — основная метрика
Слепые попарные голосования: пользователь видит два видео на один промпт без знания модели и выбирает лучшее. ELO-рейтинг (как Chatbot Arena для LLM). Единственный публичный human-preference бенчмарк для видео. T2V: 67 моделей, I2V: 50+. Обновляется в реальном времени по мере накопления голосов.
VBench / VBench 2.0 ↗
Автоматические метрики: 16+ измерений — temporal consistency, subject consistency, motion smoothness, human fidelity и др. Хорош для OSS-моделей (не требует API-доступа). Результат — % по каждому измерению. Нельзя сравнивать с ELO — измеряет другое.
Специализированные метрики
Для задач за пределами T2V/I2V единого ELO нет. LipSync: LSE-D/C (SyncNet distance/confidence); Face Swap: FID, SSIM, LPIPS; Image Animation: AED, APD; SR/FI: PSNR, SSIM. Задачи оцениваются domain-specific метриками.

Про ELO и данные
●Все ELO взяты из AA Video Arena 30.03.2026. AA показывает два вида: «Current models» (только модели с публ. API) и «All models» (все). Seedance 2.0 нет в «Current models» т.к. нет публичного API — но она есть в «All models» и лидирует там.
●T2V All models шкала: ~762 → 1 273 (Seedance 2.0). T2V Current models: Kling 3.0 #1 (1 245). I2V All models: 1 352 (Seedance 2.0 #1). Mochi 1 ELO 1 000 = точка отсчёта.
●Seedance 2.0 — #1 в обоих арен (T2V 1 273, I2V 1 352). Нет публичного API, только dreamina.com, China-first rollout Mar 2026.
●ELO живое и меняется: artificialanalysis.ai ↗
